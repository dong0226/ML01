# Sample Environment #

This is a sample environment designed in PyBullet. 

To execute the code, run the following command
```
git clone https://github.com/maple5717/ME5418-Project.git -b env_maple
```

Install the package
```
cd ME5418-Project/Object-Pushing
pip install -e .
cd ..
```

Run the test code
```
python test.py
```
